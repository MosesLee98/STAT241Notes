Random Variable

Random Var X is a function defined on the sample space S, assigning a number x = X(w) to each outcome as in the sample space


  • Random Var = Uppercase such as X and Y
  • Possible vals of random var = lowercase such as x and y

2 Types of Random Variables

  • Discrete - possible values are either finite set or an infinite sequence of values
  • Continuous - possible values are in an interval on the number line or all numbers in a collection of intervals
  • Probability Mass Function (pmf) of a discrete rv x is defined as f(x)=P(X=x) for all possible x
  • f(x) gives the probability of each possible value x of X
  • Following properties:
    1. f(x) >= 0 for all x in X
    2. summation of f(x) starting at x = 1
